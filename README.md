# Repositório de Estudos em JavaScript

Este repositório foi criado para armazenar meus arquivos e exercícios em **JavaScript**.  
O objetivo é praticar a linguagem, organizar códigos e acompanhar minha evolução enquanto aprendo programação.

---

## Estrutura
- `exercicios/` → pequenos scripts e desafios
- `projetos/` → projetos maiores ou testes de funcionalidades
- `anotacoes.md` → notas e lembretes sobre conceitos importantes
